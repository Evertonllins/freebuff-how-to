O que é o Freebuff?

O Freebuff é um agente de codificação que roda no terminal, criado pelo time da Codebuff. A proposta é simples: você instala com um comando npm, abre no diretório do seu projeto e começa a pedir mudanças de código em linguagem natural.

E como é grátis sem ser um trial? O custo é coberto por anúncios de texto exibidos no próprio CLI — a mesma lógica de apps de rádio gratuitos. Você vê um texto de ad enquanto o agente trabalha, e pronto.

por que isso importa

Ferramentas como Claude Code, Cursor ou GitHub Copilot exigem assinatura. O Freebuff permite que você experimente o fluxo de trabalho com agente de IA sem comprometer crédito ou cartão.

Instalação em 60 segundos

Você precisa ter o Node.js instalado (v18 ou superior). O comando de instalação é o mesmo em qualquer sistema — só o pré-requisito muda. Escolha o seu:

Passo 1 — Instalar o Node.js

Baixe o instalador oficial em nodejs.org e execute o .msi. Na última tela, deixe marcado "Automatically install the necessary tools" — isso garante que dependências nativas de pacotes npm funcionem corretamente.

Prefere não usar o instalador? O winget já vem no Windows 11:

powershell — como administrador
# instala Node.js LTS via winget
PS> winget install OpenJS.NodeJS.LTS

# confirma
PS> node --version
v22.x.x

Passo 2 — Liberar execução de scripts npm

O Windows bloqueia scripts de terceiros por padrão. Rode uma vez como administrador para liberar pacotes globais npm:

powershell — como administrador
PS> Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
Execution policy changed.

Passo 3 — Instalar e rodar o Freebuff

Abra o PowerShell ou Windows Terminal (sem precisar de admin desta vez) na pasta do seu projeto:

powershell ou windows terminal
# instala o freebuff globalmente
PS> npm install -g freebuff

# navega até o projeto
PS> cd C:\Users\voce\meu-projeto

# inicia o agente
PS> freebuff

✓ Freebuff pronto. Descreva o que quer fazer...

Como usar na prática

O fluxo recomendado para tarefas maiores tem quatro etapas. Para tarefas pequenas ("adicione um botão aqui", "corrija esse bug") é só digitar o pedido diretamente.

/plan

implementa

/review
Code review automático do que foi feito

Quais modelos são usados?

O Freebuff usa modelos diferentes para diferentes etapas do fluxo. Os principais são:

1

DeepSeek V4 Flash

Modelo mais rápido, usado para coleta de contexto e tarefas simples

2

Kimi K2.6 / MiniMax M2.7

Usados como agente principal de codificação — bom equilíbrio entre velocidade e qualidade

3

DeepSeek V4 Pro

O mais capaz — mas atenção: a API do DeepSeek coleta dados para treinamento do modelo deles

Brasil e o Limited Mode

O Freebuff funciona em todos os países, mas divide o acesso em dois níveis. O acesso full está disponível para uma lista de países como EUA, Reino Unido, Portugal, Alemanha e outros da Europa Ocidental. Para todos os demais — incluindo o Brasil — o serviço opera em limited mode.

Full Mode vs Limited Mode

✦ Full Mode
✓Todos os 9 subagentes
✓Múltiplos modelos (Kimi, MiniMax, DeepSeek Pro)
✓Sem limite de sessões por dia
✓EUA, UK, Portugal, Alemanha, França e outros
⚡ Limited Mode
⚠Apenas DeepSeek V4 Flash
⚠5 sessões de 1 hora por dia
✓Funciona normalmente — só com VPN também
✓Brasil, e demais países fora da lista
para usuários brasileiros

Você tem 5 sessões de 1 hora por dia com o DeepSeek V4 Flash. É menos que o full mode, mas ainda é suficiente para experimentar o fluxo completo de agente de IA — sem pagar nada. Para tarefas do dia a dia, 5 horas costumam ser mais que suficientes.

Se você usa VPN apontada para um dos países da lista de acesso full, o Freebuff também entrará em limited mode — a ferramenta detecta o modo pelo IP de saída.

⚠️ Privacidade — o que saber antes de usar

✓ O Freebuff em si não treina nos seus dados
✓ Modelos sem o label "Collects data for training" não compartilham seu código
⚠ DeepSeek V4 Pro coleta dados para treinamento — evite com código proprietário ou sensível (esse modelo não está disponível no limited mode)
⚠ O backend é na nuvem — não é uma solução 100% local
